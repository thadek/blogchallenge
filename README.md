# Warmup Challenge Alkemy

## 1) Installation
```
npm install
```

### 2) dotenv file:

```
DB_USER=
DB_PASS=
DB_NAME=
PORT=

```

#### 3) Run.
```bash
npm run dev
```

#### 3) Important:
/posts GET endpoint can receive a page query param (localhost:3000/posts?page=0)

Endpoints.
GET,POST,PATCH,DELETE
/posts /posts/:id
